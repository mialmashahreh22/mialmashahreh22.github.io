Static Analysis Tool

Vibe Sec: AI-assisted secure coding with Semgrep.

Vibe Coding Security is my VS Code extension experiment for scanning JavaScript and TypeScript code using local Semgrep rules. It connects secure coding practice with developer workflow.

  • Scan current file or full project
  • Show findings in editor diagnostics
  • Sidebar findings panel
  • OWASP-style rule baseline
